# Padantal SL — Web corporativa

Landing page de **Padantal SL**, socio estratégico global del sector pesquero:
desarrollo y operación de proyectos pesqueros internacionales, activación de capacidad
de acarreo ante ORP's, modernización de flotas e integración industrial.

## Estructura

- `index.html` — sitio completo (HTML + CSS + JS, sin dependencias de build).
- `assets/img/` — imágenes (fondo del hero).
- `assets/logos/` — logos de Padantal y de las sociedades colaboradoras.

## Desarrollo

Es un sitio estático. Basta con abrir `index.html` en el navegador.

## Despliegue (Vercel)

No requiere build. Vercel sirve `index.html` directamente como sitio estático.
